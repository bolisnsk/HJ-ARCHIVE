# HJ:ARCHIVE
인터랙션 중심의 메인 페이지 + LocalStorage 기반 CRUD 게시판 사이트입니다.
검색, 조회수, 수정/삭제 기능을 Vanilla JS로 구현했습니다

##  Link
👉[ https://your-vercel-link.vercel.app](https://hj-archive-coral.vercel.app/main.html)

##  🧩 프로젝트 소개

HJ:ARCHIVE는 저를 소개하기 위한 사이트이며, 브라우저 LocalStorage를 활용해
게시글 작성(Create), 조회(Read), 수정(Update), 삭제(Delete) 기능을 구현했습니다.
